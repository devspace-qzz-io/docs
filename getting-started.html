Getting Started

Everything you need to know before requesting a DevSpace subdomain.

What is DevSpace?

DevSpace is a curated subdomain registry that provides free username.devspace.qzz.io subdomains to developers. Unlike automated registries, every request is manually reviewed to ensure quality and prevent abuse.

Our goal is to provide developers with clean, professional URLs for their portfolios, projects, and documentation sites.

Who is DevSpace For?

DevSpace is designed for:

  • Developers who want a professional URL for their portfolio
  • Open source projects looking for a clean documentation URL
  • Side projects that need a memorable domain
  • Technical blogs and educational content
  • Demo sites for showcasing work

Your content must be developer-related. Personal blogs, business sites, and non-technical content are not eligible.

What is Allowed?

The following are permitted on DevSpace subdomains:

  • Developer portfolios and personal sites
  • Open source project documentation
  • Technical blogs and tutorials
  • Demo applications and experiments
  • Educational resources for developers
  • Tools and utilities for the developer community
Approved Hosting

Your site must be hosted on an approved platform: GitHub Pages, Vercel, Netlify, Cloudflare Pages, or Firebase Hosting.

What is Not Allowed?

The following are strictly prohibited:

  • Commercial business websites
  • Non-technical personal blogs
  • Ad-heavy or monetized content
  • Illegal or harmful content
  • Malware, phishing, or deceptive practices
  • Content that violates third-party rights
  • URL shorteners or redirect services
  • Sites primarily for SEO manipulation
Technical Restrictions

Only CNAME records are supported. A records, MX records, and custom server configurations are not permitted.

Requirements Checklist

Before submitting your request, ensure you have:

  • A live site hosted on an approved platform
  • Developer-related content
  • A public repository (for GitHub Pages) or accessible project
  • Your custom domain configured on your hosting platform
  • A GitHub account to submit the pull request
